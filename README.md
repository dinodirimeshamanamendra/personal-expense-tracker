# Personal Expense Tracker System
# Overview
The Personal Expense Tracker System is a lightweight Java-based application designed for students and individuals to manage their daily expenses easily. It helps users track where their money goes each month, categorize spending, and set simple budgets. Unlike complex or cloud-dependent apps, this system stores data locally, ensuring privacy and offline accessibility.

# Features

1) Add Expenses: Quickly log your daily expenses with categories and notes.
2) Categorize Spending: Organize expenses by type such as food, transport, entertainment, etc.
3) Set Budgets: Define monthly spending limits and monitor them.
4) View Reports: Generate simple summaries to understand spending patterns.
5) Offline Usage: Fully functional without an internet connection.
6) Data Storage: Uses MySQL database (or SQLite if you prefer) for secure local storage.

# Tech Stack

1) Language: Java
2) Database: MySQL / SQLite
3) IDE: IntelliJ IDEA / Eclipse
4) Platform: Desktop application
5) Optional: Swing or JavaFX for GUI
